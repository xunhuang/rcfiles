
# Google Doc Shortcuts

| Action                     | Shortcut   |
| ---------------------------- | ------------ |
| Increase paragraph indentation | **⌘ + ]**      |
| Decrease paragraph indentation | **⌘ + [**      |
| Clear text formatting          | ⌘ + \          |
| Insert comment                 | ⌘ + Option + m |
| Superscript                    | ⌘ + .          |
| Subscript                      | ⌘ + ,          |
| Increase font size             | ⌘ + Shift + .  |
| New Document                   | doc.new         |
