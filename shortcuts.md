
# Gmail Shortcuts

| Action                     | Shortcut   |
| ---------------------------- | ------------ |
| Back to thread list        | u          |
| Reply All                  | a          |
| Select All                 | * + a      |
| Delete                     | #          |
| Report Spam                | !          |
| Mark All as Read           | F6         |
| Send                       | ⌘ + Enter |
